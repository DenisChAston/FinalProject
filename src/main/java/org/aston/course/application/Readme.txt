1 Уровень приложения
Выполняет бизнес-логику и координирует работу между уровнями представления и домена, содержит реализацию
абстрактных определений доменного слоя, репозитории и реализации источников данных

Может содержать внутренние подпакеты: datasource (конкретные классы и сущности из пакет domain), repository, rest.

Компоненты внутри слоя application не должны зависеть от компонентов в слое presentation
