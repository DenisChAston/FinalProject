0 Уровень домена.
Уровень домена содержит бизнес-логику и компоненты, отвечающие за работу с данными, содержит определения логики функционирования приложения,
модели данных сервера, абстрактное определение объектов и определение сценариев использования.

Может сожержать внутренние подпакеты: application (сервисы для каких-то вычислений), business (Итераторы, утилиты, декораторы для работы с классами),
model (интерфейсы, абстрактные классы)

Компоненты внутри слоя domain не должны зависеть от компонентов в верхних слоях
